# Customer_shopping_behaviour_analysis
This is a data analytics project, where I analyze customer shopping behavior using SQL, Python, and Power BI. The project involves data cleaning, exploratory data analysis (EDA), SQL queries for data exploration, and interactive Power BI dashboards to identify customer purchasing patterns and generate meaningful business insights.

This project demonstrates an end-to-end data analytics workflow, starting from raw data and ending with an interactive business dashboard and presentation. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard development in Power BI, report creation, and presentation preparation.

The objective is to transform raw data into meaningful insights that support better decision-making.

Project Workflow

1. Data Loading

- Imported the dataset into Python
- Verified data structure and data types
- Examined basic dataset information

2. Exploratory Data Analysis (EDA)

- Analyzed data distribution
- Identified trends and patterns
- Created visualizations
- Generated descriptive statistics

3. Data Cleaning

- Removed duplicate records
- Handled missing values
- Corrected inconsistent data
- Standardized formats
- Prepared the dataset for analysis

4. SQL Analysis

- Imported cleaned data into SQL databases
- Executed analytical SQL queries
- Used filtering, grouping, joins, aggregate functions, and sorting
- Compared outputs across PostgreSQL, MySQL, and SQL Server

5. Power BI Dashboard

- Connected cleaned data
- Built interactive dashboards
- Added charts, KPIs, filters, and slicers
- Designed a user-friendly report layout

6. Report & Presentation

- Documented findings in a project report
- Created a professional presentation using Gamma
- Summarized key insights and recommendations

How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter notebooks and run them in sequence:
   - Data Loading
   - EDA
   - Data Cleaning
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL scripts for analysis.
6. Open the Power BI dashboard (".pbix") file to explore the visualizations.
7. Review the project report and presentation for a summary of insights.
